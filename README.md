## What is Ronin Hub?

Ronin Hub is a multi-game Roblox script hub with a custom UI built for both desktop and mobile. Instead of the usual wall of buttons, it uses proper controls — sliders, dropdowns, checkboxes, and live stat displays — so everything is actually usable.

---

## Features

- **Works on PC and Mobile** — automatically loads the right layout for your device
- **7 themes** — Dark, Light, Rosewood, Ultraviolet, Seafoam, SolarFlare, Abyss
- **Settings are saved** — sliders and preferences persist between sessions
- **Key system** — free and paid tiers available via Discord

## How to Use

1. Open the [Official Website](https://getronin.xyz) and get the loadstring.
2. Open your executor and run the loader
3. Get your key, paste it and press **Check Key**
4. The hub loads automatically for whatever game you're in

**Loader:**
```
loadstring(game:HttpGet("https://getronin.xyz/loader.lua"))()
```
